uMi_ui_scrollbox_lcl

Declaração

type TMi_ScrollBox_LCL = class(TScrollBox)

Descrição

A Classe TMi_ScrollBox_LCL foi redefinida para que os eventos DmxScroller.OnEnter e DmxScroller.OnExit sejam executados quando o ScrollBox recebe e perde o foco.

Métodos

procedure DoOnEnter(Sender: TObject);

A procedure DoOnEnter é usado para executar o evento onEnter de DmxScroller

procedure DoOnExit(Sender: TObject);

A procedure DoOnExit é usado para executar o evento onExit de DmxScroller

Constructor Create(aOwner: TComponent); Override;

O constructor Create foi redefinido para que os eventos OnEnter e OnExit sajam iniciados com DoOnEnter e DoOnExit respectivamente.

Procedure SetUiDmxScroller(aUiDmxScroller:TUiDmxScroller);

O método SetUiDmxScroller é usado para inicializar o atributo _UiDmxScroller

  • NOTA

    • Caso a propriedade DmxScroller.Active seja true, então deve-se toná-la false para que perca o vínculo com a visão anterior.

Propriedades

property UiDmxScroller : TUiDmxScroller read _UiDmxScroller write SetUiDmxScroller;

A propriedade UiDmxScroller foi crada para que os eventos DmxScroller.OnEnter e DmxScroller.OnExit sejam executados quando o ScrollBox recebe o foco e perde o foco.

  • A propriedade UiDmxScroller não deve ser published por que a mesma deve ser inicializada automatiocamente em TUiDmxScroller.SetParentLCL()
