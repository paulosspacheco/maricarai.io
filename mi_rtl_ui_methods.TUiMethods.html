class function CreateOptions(Default: LongInt;AItems: PSItem) : DmxIDstr;

A class function CreateOptions é usado para informar uma lista de opções para o campo.

  • NOTA O campo que pode receber uma lista pode ser de qualquer tipo, exceto os tipos:

    • FldEnum,FldBoolean e FldRadioButton.

  • EXEMPLO DE USO

                                                               NewSItem('Dia 15',

                                                               NewSItem('Dia 25',
                                                               nil)))))+'˜ dias˜');

  add('˜     Prazo:˜\BB'+ChFN+'Dias'+CreateOptions(2,NewSItem('30',
                                                     NewSItem('60',
                                                     NewSItem('90',
                                                     NewSItem('120',
                                                     nil)))))+'˜ dias˜');

  add('');
end;

function FMb_Bits(const aBit: Byte): Longint;

O método FMb_Bits retorna o mapa de bits da posição aBit. Ou seja: a função move o bit para a esquerda aBits posição.

  • NOTA

    • Como o mapa de bits possui 4 bytes este método gera exceção se aBit for maior que 32.

  • Example:

    • Command is: 00000100 shl 2 (shift left 2 bits)

    • Action is: 00000100 <- 00 (00 gets added to the right of the value; left 00 "disappears")

    • Result is: 00010000
